# ⚖️ Legal AI Assistant

A high-concurrency RAG (Retrieval-Augmented Generation) platform designed for legal document analysis.

### 🛠 Tech Stack
- **Backend:** FastAPI, LangChain, ChromaDB, SQLAlchemy.
- **Frontend:** React (TypeScript), Tailwind CSS, Vite.
- **AI:** OpenAI GPT-4o / Embedding Models.

### 🚀 Setup
1. **Env:** Set up `backend/.env` with your `OPENAI_API_KEY`.
2. **Backend:** `pip install -r requirements.txt`
3. **Frontend:** `npm install`
4. **Run:** Execute `./start.ps1` to launch both servers.
